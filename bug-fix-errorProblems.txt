# Bug & Error Problems Log

# บันทึก Bug และ Error ที่พบระหว่างการพัฒนา

**หมายเหตุ:** ไฟล์นี้ใช้เก็บ Log ของ Bug และ Error ที่พบระหว่างการพัฒนา โดยผู้พัฒนาจะเป็นคนบันทึกเอง เพื่อให้สามารถ Fix และ Debug ได้สะดวก

---

## วิธีใช้งาน

1. **บันทึกเมื่อเจอปัญหา:**
   - Error messages
   - Stack traces
   - สถานการณ์ที่เกิดปัญหา
   - Component/Layer ที่เกิดปัญหา

2. **ระบุข้อมูล:**
   - Date/Time
   - Component (Backend/Mobile/Admin/Arduino)
   - Environment (Development/Production)
   - Steps to reproduce

3. **บันทึกวิธีแก้ไข:**
   - Solution ที่ใช้
   - Code changes
   - Reference links (ถ้ามี)

---

## Template

```
[YYYY-MM-DD] Component: Backend/Mobile/Admin/Arduino
Error: [Error message]
Stack: [Stack trace if available]
Environment: Development/Production
Steps to reproduce:
1. ...
2. ...
3. ...

Solution:
[วิธีแก้ไข]

Status: Fixed/In Progress/Open
```
